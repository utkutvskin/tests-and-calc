## Assignment - Shapes + Calculator + Tests (Python)

Python version of the assignment. Part 1 (shapes) + Part 2 (calculator).

## Structure

```
shapes/            -> shapes implementation (Part 1)
tests/             -> unit tests (Part 1 + Part 2)
calculator.py      -> calculator (Part 2)
demo.py            -> console manual check
```

## Run tests

```
pip install -U pytest
pytest -q
```

## Run demo

```
python demo.py
```

## Notes

Part1 shapes code -> `shapes/shapes.py`
Part1 shape tests -> `tests/test_shapes.py`
Part2 calculator -> `calculator.py`
Part2 calculator tests -> `tests/test_calculator.py`
